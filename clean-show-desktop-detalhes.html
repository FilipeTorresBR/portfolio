Clean Show Desktop

Descrição

O Clean Show Desktop é um applet para o painel do Cinnamon DE em distribuições Linux, projetado para exibir a área de trabalho de maneira semelhante ao Windows. Ele se destaca pela personalização, permitindo ajustes específicos para atender às preferências do usuário.

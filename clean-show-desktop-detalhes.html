Clean Show Desktop

Descrição

Clean Show Desktop é um icone (Applet) do painel capaz de executar funções especificas no sistema para distribuições Linux que usam o Cinnamon DE que exibe a área de trabalho de forma semelhante ao Windows, porém customizável.
